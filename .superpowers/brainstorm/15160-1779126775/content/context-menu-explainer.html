What is a long-press context menu?

In iOS, a context menu appears when you long-press any item. The card blurs + scales back, and a native popup appears with labelled actions. It's the same pattern Apple uses in Mail, Photos, Safari etc.

What steps 29 + 31 + 32 add
📌 Step 29 — Long-press context menu on both card types. Three actions: Reassign, Open in Instagram, Delete

One question before designing — what happens to the inline 🗑 trash button?
A

Remove inline trash, context menu only

Cards become cleaner. Delete lives exclusively in long-press menu. Users learn the iOS pattern. Recommended — matches how apps like Notion and Apple Notes work.

B

Keep inline trash + add context menu

Trash stays visible at all times. Context menu adds Reassign + Open in Instagram. Delete can be triggered two ways. More discoverable but visually noisier.
